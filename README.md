- 原主题官网：https://ifalse.onll.cn

- 本主题由limulu24进行二次开发，增加少量功能，修改了大量样式！

## 新功能列表
- 更好看的卡片样式
- 弹出公告功能
- 广告位功能
- 自定义页脚功能
- 移动端侧边栏头部自定义功能
- 禁用wordpress版本推送功能
- 内置Font Awesome 4.7.0，FA图标库4.7.0中文官网:https://fontawesome.p2hp.com/
- 后台主题面板新样式，简洁美观
- 等等小修改......
